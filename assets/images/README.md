# Image assets

Place the classroom lesson photo provided for the hero/message background at:

```text
assets/images/hero-classroom.png
```

The CSS uses that local image first and falls back to the existing remote classroom reference if the file is not present.

## MISSION section images

Place the classroom photos used in the MISSION section at:

```text
assets/images/class01.png
assets/images/panel.png
```

These images replace the earlier SVG illustrations in the mission grid.
